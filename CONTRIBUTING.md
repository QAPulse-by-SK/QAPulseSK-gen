# Contributing to QAPulseSK-gen

Thank you for contributing! All PRs welcome.

## Getting Started

```bash
git clone https://github.com/QAPulse-by-SK/QAPulseSK-gen.git
cd QAPulseSK-gen
npm install
npm run build
```

## Development

```bash
npm run build:watch   # Watch mode
npm test              # Run tests
npm run lint          # ESLint
node dist/cli/index.js from-har examples/sample.har  # Test CLI locally
```

## PR Guidelines

- One feature or fix per PR
- TypeScript strict mode
- Update README if adding CLI options
- PRs reviewed within 48 hours
